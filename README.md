# InstaMetrics — Influencer Dashboard

A modern, responsive dashboard for Instagram influencers to track their performance metrics, engagement rates, and campaign analytics.

![Dashboard Preview](https://via.placeholder.com/800x400/0a0a0f/f0f0f5?text=InstaMetrics+Dashboard)

## Features

- **Real-time Metrics**: Track followers, engagement rates, reach, and revenue
- **Campaign Analytics**: Monitor cost per view (CPV) across different campaigns
- **Content Performance**: Analyze post types, views, likes, comments, and shares
- **Audience Insights**: Understand your audience demographics and behavior
- **Interactive Charts**: Visualize trends with sparklines and data charts
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS custom properties and gradients
- **Vanilla JavaScript**: Interactive functionality and data visualization
- **Google Fonts**: Inter font family for typography

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - this is a static website

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/lowbare3/instametrics---influencer-dashboard.git
   cd instametrics---influencer-dashboard
   ```

2. Open `index.html` in your web browser

That's it! The dashboard will load with demo data.

## Project Structure

```
├── index.html      # Main HTML file
├── style.css       # CSS styles and design tokens
└── script.js       # JavaScript functionality and demo data
```

## Features Overview

### Dashboard Sections

- **Dashboard**: Overview of key metrics and recent performance

### Key Metrics

- Total Followers
- Engagement Rate
- Cost Per View (CPV)
- Reach
- Revenue
- Likes and Interactions

## Customization

The dashboard uses demo data stored in `script.js`. To customize:

1. Update the demo data objects in `script.js`
2. Modify colors and styling in `style.css`
3. Add new sections or metrics as needed
